# Logo de ServiceFlow

Esta carpeta contiene el logo de la aplicación ServiceFlow.

## Archivos Recomendados

- `logo.png` - Logo principal en formato PNG con fondo transparente
- `logo.svg` - Logo en formato vectorial SVG (escalable)
- `logo-dark.png` - Versión del logo para modo oscuro (opcional)
- `favicon.ico` - Icono para la pestaña del navegador

## Dimensiones Recomendadas

- **Logo principal:** 512x512 px o superior (PNG transparente)
- **Favicon:** 32x32 px, 64x64 px (ICO o PNG)
- **SVG:** Sin restricciones de tamaño (vectorial)

## Uso en el Código

```tsx
// En componentes React
<img src="/images/logo/logo.png" alt="ServiceFlow Logo" />

// Para favicon en index.html
<link rel="icon" type="image/png" href="/images/logo/favicon.ico" />
```

## Notas

- Usa formatos PNG con transparencia para mejor integración con diferentes fondos
- Considera crear versiones para modo claro y oscuro
- El logo debe ser simple y reconocible en tamaños pequeños
