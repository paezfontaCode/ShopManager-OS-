# Assets e Iconos

Esta carpeta contiene recursos gráficos estáticos de la aplicación como iconos, ilustraciones y elementos de UI.

## Tipos de Archivos

- **Iconos:** Iconos de funcionalidades, categorías, etc.
- **Ilustraciones:** Gráficos decorativos para páginas vacías, errores, etc.
- **Imágenes de UI:** Fondos, patrones, decoraciones

## Estructura Sugerida

```
assets/
├── icons/          # Iconos de funcionalidades
├── illustrations/  # Ilustraciones para estados vacíos
└── ui/            # Elementos decorativos de interfaz
```

## Formatos Recomendados

- **PNG:** Para imágenes con transparencia
- **SVG:** Para iconos escalables
- **WebP:** Para imágenes optimizadas (mejor compresión)

## Uso en el Código

```tsx
<img src="/images/assets/icons/repair.png" alt="Repair icon" />
```
